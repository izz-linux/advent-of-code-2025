# Advent of Code 2025 Solutions

Solutions for Advent of Code 2025 implemented in Go.

## Project Structure

This repository contains daily solutions organized in separate directories:

- [day01](./day01)


## Quick Start

Run all solutions:
```bash
make all
```

Run a specific day:
```bash
cd day01
go run main.go
```
